## Get started

0. Get gemini api key from https://aistudio.google.com/app/apikey

1. Install dependencies
   npm install
   

2. Start the app
   npx expo start

## Run on your device

To run the app on your physical device, follow these steps:

1.  **Download Expo Go**

2.  **Start the Development Server**
    Open your terminal and run:

    npx expo start
   

3.  **Scan the QR Code**
